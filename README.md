# 🛠️ 实用工具集

纯前端在线工具集合，无需后端，直接在浏览器运行。包含数字大写转换、字数统计、智能翻译、密码生成器、密钥生成器、YAML 工具、在线中文输入法、剪贴板历史、Docker 转换器、Markdown 编辑器等 10 种在线工具，简洁高效，满足日常办公与开发需求。

## 🚀 在线使用

直接访问：https://yutheme.github.io/tools/

## 🛠️ 工具列表

### 1. 智能翻译工具
- 多语言实时翻译，支持自动语言检测
- 支持 20+ 种语言互译，智能识别，快速切换
- 使用 Google Translate 免费接口

### 2. 数字大写转换工具
- 将数字转换为符合财务规范的大写金额格式
- 支持阿拉伯数字与中文数字互转，小数与大额
- 适用于财务凭证、合同、发票等正式文档

### 3. 字数统计工具
- 精准统计文本中的各类字符数量
- 支持多种文件格式上传（TXT、DOCX、PDF）
- 实时统计，无需点击

### 4. 密码生成器
- 本地随机生成高强度密码，支持同时生成用户名+密码，字符集、长度、批量数量全配置
- **账号密码一站式生成** - 可同时生成易读用户名 + 密码，一键复制全部
- **用户名格式选项** - 支持长度自定义，可选择全小写/驼峰式(首字母小写/大写)
- **排除易混淆字符** - 自动过滤 `0/O/o` `1/l/I` 等易看错字符
- **密码强度可视化** - 直观显示「弱/中/强/极强」等级 + 暴力破解时间预估
- **常用模板一键选择** - 预置网站密码/WiFi密码/服务器密钥/支付密码/账号密码
- **批量生成** - 一次生成最多20个密码
- **可选本地历史记录** - 用户明确开启后，localStorage 保存最近生成的10条密码，完全本地存储
- **自定义特殊字符集** - 支持自定义允许的特殊符号，适配不同网站限制
- 熵值估算与一键复制，纯前端处理

### 5. 密钥生成器
- 本地生成多种格式密钥，支持 API Key、UUID、Base62、十六进制、JWT Secret 等
- **6 种密钥格式** - 覆盖 API 接口、唯一标识、加密场景等开发需求
- **自定义前缀** - 支持 `sk_` 等自定义前缀，适配各类平台规范
- **批量生成** - 一次最多生成 50 个密钥，高效便捷
- **本地历史记录** - localStorage 保存最近生成的 20 条密钥，完全本地存储
- **智能提示** - 每种格式附带使用场景说明
- 一键复制到剪贴板，纯前端处理，不上传任何数据

### 6. YAML 工具
- YAML 格式化、验证、预览一站式工具，纯前端处理
- **YAML ↔ JSON 双向转换** - 支持一键互相转换
- **格式化保留注释** - 优化解析算法保留配置注释不丢失
- **常见配置专项校验** - Docker Compose/Kubernetes 配置常见错误自动提示（ports/volumes缩进问题检测）
- **YAML 压缩** - 一键压缩成紧凑格式，方便放到配置或接口参数
- **文件上传/下载** - 支持本地 yaml/json 文件上传编辑，处理完一键下载
- **语法高亮 + 错误定位** - PrismJS 语法高亮，验证错误自动标红错误行
- **双YAML对比** - 支持两份YAML差异对比，绿色新增红色删除，快速定位配置变更

### 7. 在线中文输入法
- 基于 VRIME 的网页中文输入法，集成到工具集合
- 适合 **Meta Quest VR 浏览器** 使用，解决原生不支持中文输入痛点
- 支持拼音、双拼、五笔多种输入方案，可在设置中切换
- **完全本地运行** - RIME 引擎编译为 WebAssembly，所有输入本地处理
- 纯网页体验，无需安装，打开即可使用

### 8. 剪贴板历史记录
- 自动记录复制的文本内容，支持随时查看和复用
- **系统剪贴板监控** - 可选开启，监控整个系统的复制操作（需用户授权）
- **搜索过滤** - 快速搜索历史记录，支持关键词模糊匹配
- **一键复制** - 点击即可将历史内容重新复制到剪贴板
- **本地持久化** - localStorage 保存最多 50 条记录，完全本地存储不上传
- **去重处理** - 自动过滤重复内容，相同内容只保留最新一条
- **智能时间显示** - 显示「刚刚/分钟前/小时前」等友好时间格式

### 9. Docker 转换器
- 将 `docker run` 命令一键转换为 `docker-compose.yml` 格式
- **支持常用参数** - 端口映射（`-p`）、卷挂载（`-v`）、环境变量（`-e`）、容器名（`--name`）、重启策略（`--restart`）等
- **高级功能支持** - 网络配置、健康检查、依赖服务、特权模式等
- **5 个常用示例** - Nginx、MySQL、Redis、WordPress、完整示例快速加载
- **实时预览** - 输入时自动转换（可关闭），结果即时显示
- **复制/下载** - 一键复制 YAML 内容或下载为文件
- **解析信息展示** - 可视化展示识别到的参数，方便确认
- 纯前端处理，所有转换在浏览器完成

### 10. Markdown 编辑器
- 实时预览 Markdown，左右分屏编辑
- **实时预览** - 左侧编辑，右侧实时渲染 Markdown
- **代码高亮** - 支持 15+ 种编程语言语法高亮（JavaScript/Python/Java/Go/Rust 等）
- **导出功能** - 一键导出 .md 文件或 .html 文件
- **复制功能** - 复制原始 Markdown 或渲染后的 HTML
- **本地自动保存** - localStorage 自动保存草稿，刷新不丢失
- **文件上传** - 可打开本地 .md 文件编辑
- **同步滚动** - 左右滚动联动，预览位置自动对齐
- **字数统计** - 实时统计字符数和单词数
- **深色模式** - 支持浅色/深色主题切换
- 纯前端处理，所有数据本地存储
