# 实用工具集

一个简洁高效的在线工具集合，满足日常办公与学习需求。基于纯前端技术实现，无需服务器部署，即开即用。

## ✨ 功能特点

- 🚀 **零部署成本** - 纯静态 HTML，可直接在浏览器中打开使用
- 📱 **响应式设计** - 完美适配 PC、平板、手机等各种设备
- 🎨 **现代化 UI** - 精美的界面设计，流畅的交互动画
- 🔒 **数据安全** - 所有数据在本地处理，无需上传到服务器
- 🆓 **完全免费** - 无需注册、无广告、无收费

## 🛠️ 工具列表

### 1. 智能翻译工具
- **功能**：多语言实时翻译，支持自动语言检测
- **特性**：
  - 支持 20+ 种语言互译（中、英、日、韩、法、德、西等）
  - 智能语言识别
  - 快速语言切换
  - 一键复制结果
- **技术**：使用 Google Translate 免费接口

### 2. 数字大写转换工具
- **功能**：将数字转换为符合财务规范的大写金额格式
- **特性**：
  - 支持阿拉伯数字转换
  - 支持中文数字转换（一二三 → 壹贰叁）
  - 支持小数和大额数字
  - 实时转换，即时显示
- **场景**：财务凭证、合同、发票等正式文档

### 3. 字数统计工具
- **功能**：精准统计文本中的各类字符数量
- **特性**：
  - 统计汉字、标点、英文字母、数字
  - 支持多种文件格式上传（TXT、DOCX、PDF）
  - 实时统计，无需点击
  - 字符数、单词数多维度显示
- **场景**：文章写作、文案编辑、论文排版

## 📦 使用方法

### 方式一：直接打开（推荐）
1. 克隆或下载本项目
2. 在浏览器中直接打开对应的 HTML 文件即可使用

### 方式二：本地服务器
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server
```
然后在浏览器访问 `http://localhost:8000`

### 方式三：在线托管
可以将项目部署到 GitHub Pages、Vercel、Netlify 等免费托管平台。

## 🛡️ 技术栈

- **HTML5** - 语义化标签，良好的可访问性
- **CSS3** - Flexbox、Grid 布局，CSS 动画
- **Vanilla JavaScript** - 原生 JS，无框架依赖
- **Tailwind CSS** - 部分工具使用 Tailwind CSS 进行样式构建
- **Font Awesome** - 图标库
