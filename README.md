## 🛠️ 工具列表

### 1. 智能翻译工具
- 多语言实时翻译，支持自动语言检测
- 支持 20+ 种语言互译，智能识别，快速切换
- 使用 Google Translate 免费接口

### 2. 数字大写转换工具
- 将数字转换为符合财务规范的大写金额格式
- 支持阿拉伯数字与中文数字互转，小数与大额
- 适用于财务凭证、合同、发票等正式文档

### 3. 字数统计工具
- 精准统计文本中的各类字符数量
- 支持多种文件格式上传（TXT、DOCX、PDF）
- 实时统计，无需点击

### 4. 密码生成器
- 本地随机生成安全密码，支持字符集与长度配置
- 熵估算与一键复制，纯前端处理

### 5. YAML 校正预览器
- 前端解析与格式化 YAML，提供校正建议与验证
- 支持常见数据类型与嵌套结构，错误行提示

## 📦 快速开始

1. 克隆或下载本项目
2. 在浏览器中直接打开对应的 HTML 文件即可使用

## 🛡️ 技术栈

- **HTML5** - 语义化标签，良好的可访问性
- **CSS3** - Flexbox、Grid 布局，CSS 动画
- **Vanilla JavaScript** - 原生 JS，无框架依赖
- **Tailwind CSS** - 部分工具使用 Tailwind CSS 进行样式构建
- **Font Awesome** - 图标库


## 🌟 特色功能

### PWA 支持
项目已配置 PWA 清单文件，支持添加到主屏幕，提供类原生应用体验。

### SEO 优化
- 语义化 HTML 标签
- 合理的 meta 标签配置
- 良好的页面结构

### 性能优化
- CSS 变量统一管理主题
- 避免不必要的 DOM 操作
- 使用防抖优化输入事件
- 图片资源懒加载

## 📝 注意事项

### 翻译工具
- 使用的是 Google Translate 免费接口
- 请求频率可能有限制，建议避免大量频繁请求
- 适合个人或小型项目使用

### 浏览器兼容性
- 推荐使用现代浏览器（Chrome、Firefox、Edge、Safari 最新版本）
- IE 浏览器不完全支持
